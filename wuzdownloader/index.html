Basic Instalation

To open and build the app you need to set up an Android studio.

  • Download Android Studio - https://developer.android.com/studio/
  • Install Android Studio - https://developer.android.com/studio/install
  • Learn more about Android Studio - https://developer.android.com/studio/intro/

App Configuration

Import Project in Android studio


Step 1:Select ‘Import Project’ option and find your “wuz_downloader" Project





Change package name


Step 1: Throughout these steps, I will use com.yourapp.android as a placeholder for your app's package name that you must replace with yours.

Step 2: In the Project view on the left, click the Gear icon, and make sure Compact Middle Packages is unchecked (turned off).

Step 3: In the Project view, now expand app > java until you see com.example.wuzdownloader package.

Step 4: Right-click on example, choose Refactor > Rename option and enter the replacement for "example" part i.e., yourapp in our example case. Repeat this process to rename wuzdownloader to yourapp. Note: When renaming com in Android Studio, it might give a warning. In such case, select Rename All

Step 5:

Find build.gradle file inside app folder and click to open and change your applicationId. You may need to change the package= attribute in your manifest, Clean & Rebuild project

Change app name


Open app gradle file and change your app_name here

Change font


Go to app/src/main/../assets/fonts and change ttf file



And open res/values/string.xml to change fonts path


Change color

Go to res/values/color.xml and change the colors

Firebase Configuration


Create Firebase Project

Before you can add Firebase to your project, you need to create a Firebase project to connect to your app. Visit Understand Firebase Projects to learn more about Firebase projects.

Visit for more information on how to add firebase to your project

Register App to Firebase

Firebase console's project overview page, click "Add Project" to launch the setup workflow.



Check "Enable Google Analytics for this project" to enable analytics.




Add google-services.json

Click Download google-services.json to obtain your Firebase Android config file (google-services.json). Move your config file into the android/app/src directory of your project. You can select the Project view in the top-left corner of the Project navigation view, and then open the app directory. You can then drag the google-services.json file into the Android Studio project. You should end up with a project file tree like the following screenshot.



Make sure that you enter the ID that your app is actually using. You cannot add or modify this value after you register your app with your Firebase project. Both application ID should be matched.
